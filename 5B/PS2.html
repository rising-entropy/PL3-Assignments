Program to obtain count of digits in a number

Problem Statement

Write a code in any programming language that counts the digits in a number



Code

We write the program in Python

# We write the recursive call function.

def countDigit(n):
  if n/10 == 0:
    return 1
  return 1 + countDigit(n // 10)

n = int(input())

# Calling the Function in main
print(countDigit(n))


Testcases

Sample Input

68328

Sample Output

5

Sample Input

2

Sample Output

1


Variables

n - We shall traverse the recursive function through this n that increases the count for every number encountered.


Complexity

Time Complexity: O(n)



Alternate Method

Using the Logarithm

We shall make use of the property of log10 n
Essentially a number with 104 degree value can hence be identified to have a count of 4 digits.
